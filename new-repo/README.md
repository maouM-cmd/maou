# New Repository Template

このディレクトリは「新しいリポジトリ」として使える最小構成のテンプレートです。

## 含まれるファイル

- `main.py`: 動作確認用のサンプルスクリプト
- `.gitignore`: Python向けの基本的な除外設定

## 使い方

```bash
cd new-repo
git init
git add .
git commit -m "Initial commit"
python3 main.py
```

## 実行結果

`python3 main.py` を実行すると、リポジトリ作成完了メッセージを表示します。
