# 🐦 シマエナガAIニュースダッシュボード

🌐 **[▶ アプリを開く](https://nvqu6jlxz9yzy38m4cm42j.streamlit.app/)**

もふもふシマエナガと一緒に最新のAIニュースをチェック！

## 🎄 特徴

- 🔍 Google News RSSからAIニュースを自動収集
- 🐦 シマエナガ×クリスマステーマのかわいいデザイン
- 📱 カード型で見やすいレイアウト
- ✨ ホバーエフェクトでインタラクティブ

## 🚀 使い方

1. サイドバーでキーワードを入力
2. Enterで検索
3. カードをクリックで記事へ

## 📦 ローカル実行

```bash
pip install streamlit feedparser
streamlit run app.py
```
